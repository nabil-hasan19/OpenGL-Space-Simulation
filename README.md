# 🚀 OpenGL Space Simulation

A 3D Space Simulation developed in **C++ using OpenGL (GLUT & GLU)** that demonstrates the basic concepts of Computer Graphics including hierarchical transformations, orbital motion, animation, and user interaction.

The simulation contains a realistic solar system scene where the Sun rotates, Earth revolves around the Sun, the Moon revolves around the Earth, and a rocket launches a satellite into orbit through multiple animation stages.

---

## 📌 Project Features

- 🌞 Rotating Sun
- 🌍 Earth Orbit around the Sun
- 🌙 Moon Orbit around the Earth
- 🚀 Rocket Model
- 🛰 Multi-stage Satellite Launch Animation
- ⭐ Random Star Background
- 🎥 Static and Dynamic Camera Views
- 🖱 Right-click Interactive Menu
- ⌨ Keyboard Controls
- 🧩 Hierarchical Transformations
- 🔄 Real-time Animation using GLUT Timer

## 🛰 Satellite Launch Stages

The satellite deployment is divided into three phases:

1. **Deployment**
   - Satellite moves vertically from the rocket.

2. **Translation**
   - Satellite travels horizontally away from Earth.

3. **Orbit**
   - Satellite starts revolving around Earth.
